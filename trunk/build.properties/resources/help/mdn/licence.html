Licence Activation

You must register and licence Mobile Data Now before you can use the product. The licence agreement and this licence activation screen will be displayed whenever you start the MDN Server if the product is not licensed.
See the Installation and Configuration guide for more information on installing, licensing and starting the MDN Server and Licence Activation screen.

To register and licence Mobile Data now you must:

1. Start the Licence Activation screen.
2. Register Mobile Data Now.
3. Enter the Activation Key.


Start the Licence Activation Screen

The Licence Activation screen is displayed whenever you start the MDN Server if MDN is not yet licensed. You may also manually start the licence activation screen by selecting it from the Start|Programs|Mobile Data Now menu (under Windows), or type licence at a Linux/Solaris command prompt.

Start the Licence Activation screen and record the Customer Licence Key for later use when you register MDN.

The Customer Licence Key is generated by MDN and will be need to be entered when you register MDN.

When you have registered MDN you will be provided with an Activation Key which is entered in this field.

Product ID is a number identifying the MDN product and version.

Licence Duration is the date when the evaluation version expires in YYYYMMDD format, or Permanent for a full licence.

Num Users is the number of simultaneous on-line WAP clients that may use MDN.


Register Mobile Data Now

To register MDN go to the Mobile Data Now section on the WAP Solutions web page at www.wapsolutions.co.nz and follow the instructions to register MDN.

When you register you will need to enter your company details and the Customer Licence Key generated by the licence activation screen. WAP Solutions will then send you the Activation Key.


Enter the Activation Key

When you register MDN WAP Solutions will send you an Activation Key to licence and activate Mobile Data Now. Enter the activation key in the Activation Key field of the Licence Activation screen and press the Set Key button.
If the activation key is valid the Product ID, Licence Duration and Num Users fields will be filled in with the values appropriate to the activation key. Invalid activation keys will be rejected.
