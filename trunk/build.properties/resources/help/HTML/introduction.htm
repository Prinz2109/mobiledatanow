MobileDataNow lets you connect to databases and web services, then access that data using messaging (email, SMS text messaging or instant messaging) via preset queries.

 

Your users send a message with their query and MobileDataNow sends the results back.

 

 

This configuration guide aims to quickly get you working with MobileDataNow by taking you through a simple setup to extract data from a database:

 

In this tutorial we will setup a simple query to extract information from a database

For example: Sending a message to find all contacts beginning with a certain letter of the alphabet. 

This should take no longer than 10 - 20 minutes to complete

 

We will:

 

- Install the software

- Create a new Project

- Link to a database and decide which tables and fields to use by creating a View

- Add a user and assign that user to a group, then assign access privileges to that group

- Create a group for public messaging

- Setup a query for the user, test it. Decide the keywords used to access the information, setup a messaging center, decide the format of the returned information and test it.

- Deploy to our user and access the information using the query we just created

 

Navigate through each chapter using the menu at the left.
