Instant Messaging:

Send an IM to the base IM address (this will show up as a Friend/Contact in your IM application) with the words 'cont h'. This will query the database for all contact names starting with the letter H

Note: You will need to do this from a different computer than the host computer running MobileDataNow as IM networks only allow one connection per computer running at a time.

 

You should receive a reply back like below

 

 

SMS:

Send an SMS to the base SMS number with the letter 'r' in the body of the text message. This will query the database for all contact names starting with the letter R, but since we opted to only return 1 result (in the messaging details screen) only 1 result will be returned.
