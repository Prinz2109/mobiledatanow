Overview of the user interface

After you login, a blank project tab will already be open.

 

MobileDataNow uses a simple work flow. Your projects are listed in the Manage Projects button at the top of the screen, and as you open projects, the projects are listed as tabs along the top of the user interface.

 

Within each Project, work your way across, with..

Data Sources, then Users & Groups and finally Queries.

 

Each time you want to add a database, view, user, group or other element, press the button in the tree according to the element you wish to add.

Unwanted elements, like Views or Users can be dragged to the recycle bin, and also restored by clicking on the recycle bin.

 

Global settings are configured at the top right of the screen with the button Settings.

 

If you want to add more users, click the Buy button which will give you reseller information.

 

 

 

 

The buttons at the top all open windows which can be moved around as you work.
