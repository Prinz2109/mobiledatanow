Login

The Login screen allows you to log in to the application to verify your identity and access privileges. To login you must supply a valid user name and password. For security the password you enter is not displayed in a readable format.
If the user name or password is incorrect you will not be permitted access to the system.

Once you have entered your user name and password pres the Login button to login and start using the application.

Press Exit to exit he application if you decide not to login.

Contact the system administrator if you need to create an account, change your access privileges or reset your password.
