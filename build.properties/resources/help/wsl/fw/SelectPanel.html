Search

The Search screen allows you to select a specific record from a list of available records. When it is first displayed the Search screen will often display a full list of all the available records.

If no records are displayed or if the list is too long to easily find the desired record you can enter the first part of the record name in the Name field and press the Search button. The matching records will be displayed in the Results panel. If the Name field is empty all records will be selected.

Choose a record by clicking on it with the mouse and then pressing the Select button. Press Cancel if you decide not to select a record.
