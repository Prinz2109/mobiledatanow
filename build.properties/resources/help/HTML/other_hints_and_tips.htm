My MDN Account

 

In the Settings screen, you can view your license details and change your registered email address.

 

 

 

Language options

 

You can translate the user interface of MobileDataNow in to another language. Japanese, Chinese, German etc. Download the Gui_en.xml file from the C:\Program Files\MobileDataNow\tomcat-5.5.25\webapps\dashwell.war\lang directory and start making changes. You can upload your new language file in to MobileDataNow by going to Settings, then Language after logging in.

 

 

 

Recycle Bin

 

The recycle bin is located at the bottom of each section in MobileDataNow. Using your mouse, drag and drop any unwanted Projects,  Database Connections, View, Users, Groups and Messaging Queries to the Recycle Bin.

 

 

Click on the Recycle Bin to open it and click Restore to retrieve any elements again, or Delete them permanently.

 

 

Note: If you've removed any unwanted elements to the Recycle Bin, you should delete them permanently. When elements are removed to the recycle bin, they are only given a different status in the database and may conflict with equivalent newly created elements to give you an error message about a duplicate. If you delete the element permanently, you won't have these problems.

 

 

Context Sensitive Help

 

Click the question mark icon at the top right of the screen, a small help screen drops down. Now move your mouse over each title or heading and hints and tips are displayed about what to enter in the corresponding field.

 

 

 

Multiple project tabs

 

You can easily open or create several projects at once and move between them. Settings and Users are common to each project to save you time configuring them for each project. Even though users are common to all projects, if users are not added to a group, they will not have permissions to the queries created.
