Mobile Data Now Web Help

Mobile Data Now Web (MDN Web) is the HTML (or Web) interface to the Mobile Data Now Application. It allows you to use Mobile Data Now with any Web browser. This includes Web Browsers for Palm Pilots, Pocket PCs and other PDAs as well as standard Web Browsers for you Desktop PC.


Title Bar

The main Title Bar at the top of every Mobile Data Now Web screens contains a title and 3 links: Menu, Logout and Help.
The Menu link will bring up the MDN Web Menu screen (in its last used state). The Logout link will log you out of MDN Web. The Help link will bring up this document at the section that pertains to the current context of the application.


Splash Screen

The Splash Screen is the Welcome screen that appears when you go to the Mobile Data Now Web URL, using a bookmark or just by typing the URL.
The Login Settings in the Presentation Builder of Mobile Data Now Administrator allows you to customise the text on this screen or to bypass it altogether.


Login Screen

This screen allows you to log into MDN Web. The Login Settings Properties screen in the MDN Administrator Presentation Builder allows you to specify the text and settings for this screen.


Logout Screen

The logout screen appears when you have logged out of MDN Web. Click the Login button to log back into MDN Web.


Menu Screen

The Menu screen shows your Mobile Data Now menu as an Explorer like tree. This allows you to easily navigate to the different menu items made available to you in MDN Administrator Presentation Builder. Note that different types of menu item have different icons associated with them.
If the menu is not showing due to another screen being shown you can click the Menu link in the Title Bar to make it reappear.


Query Parameters Screen

This screen allows you to enter the values for a parameterised query. Enter the required values and click the Query button to execute the query and display the query results.


Query Results Screen

This screen shows the results of a Query as a list of records. This list shows up to 20 records at a time. Click the More link at the end of the list to get the next screen of results. If you click on one of the records in the list the properties of this record will be displayed in the Show Record screen


Show Record Screen

This screen shows you the properties of a record that has been selected in a query results list. You are able to delete the record by clicking the Delete Record link under the properties. You are able to edit the record by clicking the Edit Record link under the properties.


Delete Record

If you click the Delete Record link on a Show Record screen then you will be prompted for confirmation of the delete. If you click the Delete button on the confirmation screen the record will be deleted and you will be returned to the menu.


Edit Record Screen

You may edit a record by clicking the Edit Record link on the Show Record screen. The Edit record screen allows you to change the values of the properties of a record. To save the changes you have made click the Save button. To cancel the changes click the Back button of your browser or the Menu link in the title bar


Insert Record Screen

The New Record screen allows you to create a new record for a view. To save the new record you have created click the Save button. To cancel the changes click the Back button of your browser or the Menu link in the title bar


Error Screen

This screens shows any errors that occur during the operation of MDN Web. To return to MDN Web click the back button or the Menu link in the title bar.
